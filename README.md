# Lexicon — Dictionary App

A React dictionary app powered by the [Free Dictionary API](https://dictionaryapi.dev/).

## Features
- Search any English word
- Definitions grouped by part of speech
- Audio pronunciation
- Example sentences
- Clickable synonyms & antonyms

## Tech Stack
- React + Vite
- CSS Modules
- Free Dictionary API

## Getting Started

```bash
npm install
npm run dev
```

## Deploy on Netlify

```bash
npm run build
```
Then drag the `dist/` folder to [netlify.com](https://netlify.com).
